# ChromeBlocker (MV3)

Local Chrome extension that blocks websites on a schedule (two windows per day, per weekday).

## Install (developer mode)

1. Open Chrome → `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this folder: `c:\Users\_chromeblock`

## Use

- Open the extension → **Open settings**
- Add domains like `reddit.com`
- Enable days and set your time ranges
